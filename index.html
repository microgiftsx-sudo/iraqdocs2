shield البيانات التي نجمعها

نحن نحترم خصوصيتك ونلتزم بحماية بياناتك الشخصية. المولد العراقي الذكي يجمع البيانات التالية:

  • مفتاح API: يتم تخزينه بشكل آمن في إعدادات Google Docs الخاصة بك
  • بيانات المستند: النص الذي تدخله لتوليد المستندات
  • معلومات الاستخدام: لتحسين الأداء وتطوير الخدمة

lock كيف نحمي بياناتك

نستخدم تشفير SSL لنقل البيانات بشكل آمن. مفتاح API الخاص بك لا يتم مشاركته مع أي طرف ثالث أبداً.

cloud استخدام Google Gemini API

الأداة تستخدم Google Gemini API لتوليد المستندات. بياناتك تُرسل إلى Google لمعالجة الطلبات فقط. راجع سياسة خصوصية Google للمزيد من التفاصيل.

storage التخزين والاحتفاظ بالبيانات

لا نقوم بتخزين بياناتك على خوادمنا. جميع البيانات تُحفظ محلياً في Google Docs الخاص بك.

verified_user حقوقك

لديك الحق في:

  • الوصول إلى بياناتك وتعديلها
  • حذف مفتاح API في أي وقت
  • إيقاف استخدام الأداة تماماً

info قبول الشروط

باستخدامك للمولد العراقي الذكي، أنت توافق على الالتزام بهذه الشروط والأحكام. إذا لم توافق، يرجى عدم استخدام الأداة.

description الغرض من الأداة

المولد العراقي الذكي هو أداة مساعدة لتوليد المستندات الرسمية العراقية بشكل أسرع وأسهل. النتائج المُولدة قد تحتاج إلى مراجعة وتعديل من قبل المستخدم.

warning مسؤوليات المستخدم

أنت مسؤول عن:

  • مراجعة جميع المستندات المُولدة قبل استخدامها
  • ضمان دقة وموثوقية المعلومات المدخلة
  • الامتثال للقوانين والأنظمة العراقية المعمول بها
  • حماية مفتاح API الخاص بك وعدم مشاركته

block الاستخدام الممنوع

يمنع استخدام الأداة لأغراض:

  • غير قانونية أو مخالفة للقوانين العراقية
  • تزوير المستندات أو تقديم معلومات كاذبة
  • انتهاك حقوق الغير أو ممتلكاتهم الفكرية
  • أي أغراض ضارة أو غير أخلاقية

gpp_maybe الضمانات

الأداة تُقدم "كما هي" دون أي ضمانات، سواء صريحة أو ضمنية. نحن لا نضمن:

  • أن الأداة ستلبي جميع متطلباتك
  • أن العمل سيكون بدون انقطاعات أو أخطاء
  • دقة أو جودة المستندات المُولدة

update التحديثات والتغييرات

نحتفظ بالحق في تعديل أو تحديث هذه الشروط في أي وقت. سيتم إشعارك بأي تغييرات جوهرية من خلال الأداة.

email التواصل معنا

لأي استفسارات أو شكاوى، يمكنك التواصل معنا من خلال:

  • قسم "إعدادات مفتاح API" في الأداة
  • repository الخاص بالمشروع على GitHub

vpn_key كيف أحصل على API Key؟

اتبع الخطوات البسيطة التالية للحصول على مفتاح Gemini API مجاناً في دقيقة واحدة:

1 انتقل إلى Google AI Studio
افتح الرابط التالي في نافذة جديدة:
open_in_new https://aistudio.google.com/app/apikey
2 سجل الدخول بحساب Google
استخدم حساب Gmail الخاص بك. إذا لم يكن لديك حساب، يمكنك إنشاء حساب مجاني في دقيقة واحدة.
3 أنشئ مفتاح API جديد
في صفحة Google AI Studio، ابحث عن زر "Create API Key" أو "إنشاء مفتاح API" واضغط عليه.
4 انسخ المفتاح
سيظهر لك المفتاح بهذا الشكل:
AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX اضغط على أيقونة النسخ بجانب المفتاح.
5 أدخل المفتاح في الأداة
ارجع إلى هذه الأداة، الصق المفتاح في حقل "Gemini API Key" ثم اضغط "حفظ المفتاح".
info معلومات مهمة
  • • المفتاح مجاني تماماً للاستخدام الشخصي والتجاري الصغير
  • • Google تقدم كريمات مجانية كافية للاستخدام العادي (15 طلب/دقيقة)
  • • لا نشارك مفتاحك مع أي طرف ثالث - يُحفظ محلياً فقط
  • • يمكنك حذف المفتاح في أي وقت من إعدادات Google Cloud
  • • الأداة تعمل بدون اتصال بالإنترنت بعد التوليد الأول
